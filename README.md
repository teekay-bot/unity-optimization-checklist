# Unity 6 URP — Optimization Checklist

A step-by-step new-project optimization checklist for **Unity 6.3 LTS** with the **Universal Render Pipeline**.

## [→ Open the Checklist](https://teekay-bot.github.io/unity-optimization-checklist/)

## What's inside

- Player Settings, Quality Settings, URP Asset configuration
- Batching strategy with SRP Batcher explanation
- Per-tier performance budgets (PC Low → Mobile High)
- Asset optimization (textures, meshes, lighting, shaders, audio)
- Memory & GC best practices
- Pre-release build verification
- Profiling & device testing workflow

Checkbox progress is saved locally in your browser via `localStorage`.
